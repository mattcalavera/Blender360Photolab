# Blender360Photolab v0.29.0-beta

This is the first public beta release of **Blender360Photolab**, a Blender add-on for working with equirectangular 360° photos.

The add-on provides a practical workflow for importing, previewing and reframing 360 images directly inside Blender, with support for cylindrical previews and rectilinear PNG export.

---

## Highlights

- 360° equirectangular photo workflow inside Blender.
- Camera-based reframing.
- Cylindrical preview generation.
- Rectilinear preview and export workflow.
- Compact preview layout.
- CIL and CLV previews arranged on the same upper row.
- Rectilinear preview centered below.
- PNG export.

---

## Recommended Use

Use this release if you want to test Blender360Photolab with:

- drone 360 photos;
- panoramic photography;
- architectural visualization;
- virtual tour preparation;
- creative reframing of equirectangular images.

---

## Installation

For this beta release, the recommended installation method is the single Python file if provided in the release assets.

Install from:

```text
Edit > Preferences > Add-ons > Install from Disk
```

Do not use GitHub's automatically generated “Source code ZIP” unless it is explicitly marked as installable.

## Known Limitations

- This is a beta release.
- Compatibility has not yet been tested on every Blender version.
- Documentation is still minimal.
- Some workflow steps may still require manual adjustment.
- Additional projection/export tools may be added in future releases.
