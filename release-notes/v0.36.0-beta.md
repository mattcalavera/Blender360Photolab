# Blender360Photolab — Beta Release

This beta release introduces **Blender360Photolab**, a Blender add-on for reframing equirectangular 360° still photos into conventional printable images.

The add-on is designed for photographers, drone users, panoramic workflows, and anyone who needs to extract controlled photographic views from high-resolution 360° images directly inside Blender.

## Main features

- Reframe **equirectangular 360° still photos** inside Blender.
- Automatic scene setup after loading an image.
- Automatic **Rendered Camera View** for the main framing workflow.
- Projection modes:
  - **RET** — Rectilinear / perspective view.
  - **CIL** — Cylindrical projection.
  - **CLV** — Level cylindrical projection, designed to reduce horizon bending by treating pitch as a vertical shift.
- Comparative preview layout:
  - **CIL** and **CLV** on the top row.
  - **RET** centered below.
- Orthographic preview view without the semi-transparent camera frame.
- Direct export without relying on Blender’s normal render output.
- Export formats:
  - **PNG 8-bit lossless**
  - **TIFF 16-bit lossless**
  - **High-quality JPEG**
- `.cube` LUT support.
- LUTs are automatically applied to previews and final exports when loaded.
- Print-oriented resolution tools:
  - preset output sizes;
  - custom output resolution;
  - estimated print size at 150 DPI and 300 DPI;
  - native/detail-aware resolution estimation for the current framing.
- Robust equirectangular sampling with bilinear interpolation.
- Seam handling for the ±180° equirectangular boundary.
- Fix for Blender add-on installation/registration issues caused by accessing `bpy.data` during `register()`.

## Installation

For Blender **4.2+ / 5.x**, install the add-on as a Blender extension package.

Download the ZIP file attached to this release under **Assets**.

Do **not** install GitHub’s automatically generated:

- `Source code (zip)`
- `Source code (tar.gz)`

Those files are repository archives and are not the Blender extension package.

Use the release asset ZIP instead, the one containing the add-on files and `blender_manifest.toml`.

Then in Blender:

1. Open **Edit > Preferences**.
2. Go to **Get Extensions** or **Add-ons / Extensions**, depending on your Blender version.
3. Choose **Install from Disk**.
4. Select the ZIP file downloaded from the release assets.
5. Enable **Blender360Photolab**.
6. Open the **3D Viewport Sidebar** and use the **Foto 360** panel.

## Workflow overview

1. Load an equirectangular 360° photo.
2. Compose the image using Camera View.
3. Choose the projection mode: RET, CIL, or CLV.
4. Generate previews to compare the projection behavior.
5. Optionally load a `.cube` LUT.
6. Choose export resolution and format.
7. Export the final reframed image.

## Known limitations

This is a beta release.

- The add-on is focused on **still images**, not video reframing.
- CLV helps reduce horizon curvature, but it cannot mathematically remove all projection distortion in every case.
- Very wide fields of view may still show visible projection stretching.
- TIFF export is intended as a high-quality master format, but file sizes can be large.
- LUT handling is practical and export-oriented; full color-management workflows may still require external grading tools.
- Performance depends on source image resolution, output resolution, and available system memory.
