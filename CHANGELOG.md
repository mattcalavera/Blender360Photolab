# Changelog

All notable changes to **Blender360Photolab** will be documented in this file.

This project follows a simple release history model. Internal development versions may be summarized only when they become relevant to a public release.

---

## [0.29.0-beta] - 2026-05-03

### Added

- First public beta preparation.
- Project name finalized as **Blender360Photolab**.
- Compact preview layout for the 360 photo workflow.
- CIL and CLV cylindrical previews arranged on the same upper row.
- Rectilinear preview centered below the cylindrical previews.
- PNG export workflow for lossless output.

### Changed

- Public-facing documentation prepared for GitHub release.
- The internal V029 baseline is considered the reference version for the first beta publication.

### Fixed

- Improved transition behavior between CIL and CLV preview views.
- Reduced view jumps while switching preview modes.
- Improved camera/view consistency during the preview workflow.

### Notes

- This is a beta release.
- Some workflow steps may still require manual adjustment inside Blender.
- Compatibility testing across multiple Blender versions is still in progress.

---

## [0.28.0] - Internal development version

### Added

- Base 360 photo reframing workflow.
- Equirectangular image handling.
- Camera-based reframing.
- Initial cylindrical preview generation.
- Initial rectilinear output workflow.

---

## Earlier versions

Versions before `0.28.0` were internal development builds and are not intended for public release.
