# Security Policy

## Supported versions

360 PhotoLab is currently in beta.

Security-related reports should refer to the latest public release unless the issue affects a specific older version.

---

## Reporting a security issue

If you believe you have found a security issue, please do not publish exploit details publicly before maintainers have had time to evaluate the report.

Open a private contact channel if available, or create a GitHub issue with minimal public details and clearly mark it as security-related.

Please include:

- affected version;
- Blender version;
- operating system;
- clear description of the issue;
- steps to reproduce, if safe to share;
- potential impact.

---

## Scope

Relevant issues may include:

- unsafe file handling;
- unexpected execution of external commands;
- insecure handling of user-provided paths;
- behavior that could overwrite user files without clear confirmation.

General bugs, crashes or UI issues should be reported as normal GitHub Issues.
