# Community Post Drafts

## Blender Artists

```markdown
Hi everyone,

I’m sharing the beta release of **360 PhotoLab**, an add-on for Blender for reframing equirectangular 360° still photos into conventional photographic exports.

It provides a camera-based workflow with RET/CIL/CLV projection modes, automatic Rendered Camera View setup, comparative previews, `.cube` LUT support, print-size estimation and PNG/TIFF/JPEG export.

The add-on is mainly intended for drone panoramas, 360° photography, virtual tours, landscape images and print-oriented reframing workflows.

GitHub:
https://github.com/mattcalavera/360PhotoLab

Latest release:
v0.36.0-beta

I’d be very interested in feedback about Blender 4.2+ / 5.x installation, projection behavior, LUT workflow and large-image export performance.
```
